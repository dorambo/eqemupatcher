Installation:

1. If reinstalling or upgrading the UI from a previous install, delete the "Defiance" folder located in your "EverQuest\uifiles" directory.

2. Extract the "Defiance" folder from the zip archive to your EverQuest "uifiles" directory.


Activation:

1. In game, open the "options" menu. On the "General" tab click the "Load UI Skin" button. Click "Defiance" from the list of skins. (Optional) Enable "Keep Your Layout" button. Click "Load Skin".


(Optional) Set Transparency:

1. To achieve the same transparency shown in the interface screenshots:
	1.1. To set all interface windows transparency: While in game, open the "options" menu. On the "Display" tab change the "Window Transparency" and the "Fade-to Transparency" sliders to 75%.

	1.2. To individually set interface window's transparency: While in game, right click the window you want to adjust. Mouse over "Window" then "Alpha", set "Normal Level" and "Faded Level" sliders to 75%.
